### 搭建环境

参见README.md以及makefile文件即可

1. 新建一个虚拟环境

   ```
   virtualenv.exe flaskbb_master_env
   activate.bat
   ```

2. make install

   ```
   pip install -r requirements.txt
   clear
   pip install –e .
   flaskbb install
   ```

3. make run

   ```
   flaskbb  --config  flaskbb.configs.development.DevelopmentConfig run
   ```
